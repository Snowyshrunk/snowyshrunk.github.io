Bot Setup Complete

Your server has been configured with the bot. Below are all the available features and how to use them.

🎂Birthday Features

Manage and celebrate birthdays across all servers where the bot is present.

/set-birthday <date> Set your own birthday (DD/MM format: Day/Month)
  • This command can be re-run to update the date if you would like
  • Birthdays are announced automatically in general
  • Birthdays sync between servers - if you've added your birthday in one server, it will work in all servers this bot is in

🚫Blacklist Features

Protect your server by detecting and managing blacklisted users.

/check-blacklist Check for blacklisted users in this server (Admin only)

Auto Ban/Detect

  • When a blacklisted user joins, you will be notified in the logs channel
  • Any users with the administrator permission can use the ban button on the notification

✅Verification Features

Set up a verification system to protect your server from unwanted users.

/verify Assign yourself the verified role (@verified)
/preview_verification_messages Preview the DMs sent to users (Admin only)
/notify-unverified Send DMs to unverified users (Owner only, one-time)

This is to aid in the setup of the verification system

/assign-nonverified Assign all people without the verified (@verified) role the non-verified role (@not verified) (Admin only)
  • New members will receive DMs with verification instructions

🎭Reaction Roles

Allow users to assign themselves roles by reacting to messages.

/setup_reaction_roles <channel> <message_id> <emoji> <add_role> [remove_role] Configure reaction roles (Admin only)
  • Users can react to the specified message to get/remove roles

🔢Counting Game

A fun counting game where users count sequentially in a designated channel.

/setup_counting <channel> Set the counting channel (Admin only)
  • Users count up from 1 in the designated channel
  • Automatic reset on wrong numbers with forgiveness system
  • Note: To change the counting channel, re-run /setup_counting

📋General Commands

Additional commands for bot configuration and management.

/setup Reconfigure bot settings (Admin only)
/view_config View current configuration
  • All commands respect the enabled features for this server

💡Suggested Setup

Recommended Configuration:

It is recommended to restrict non-verified members to not be able to send messages in all but one channel called 'verification' (or something similar) where they can run the /verify command. The channel should not be visible to verified users and should include a pinned message telling users to verify.

I personally suggest removing any access to view all channels other than announcement and information channels (e.g., rules).
